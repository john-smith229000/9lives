# The Task Ladder — day by day, what each task pays toward the goal

Every task, and exactly what it contributes, so nothing reads as busywork. Companion to
`quest_web.md` (the "five currencies" summary) and the `../story/day*.md` beats. Roles only.

## The goals & the running shopping list

- **G1 — relight the beacon.** Fills a checklist across the game:
  `☐ lens · ☐ oil · ☐ usable stair · ☐ tower key · ☐ flame`.
- **G2 — clear the berth.** One physical job (Day 6).
- **G3 — comfort the friend (and the partner).** Ongoing; **never a cure** — only the ship's
  vet cures. This is agency + heart, fed by the fish supply you fix.

**Currencies:** **Info** (what/why/where) · **Access** (reach the next task) · **Part** ·
**Fuel/Tool** · **Goodwill** (opens people/places) · **Comfort** (feeds G3).

## Reading the ladder
- **Days 1–3 are gathering** (info → the crates → the parts/fuel/tools).
- **Days 4–5 fix the *supply* line** (fish → G3) and **open the keeper** (Goodwill).
- **Day 6 finishes G2**, **Day 7 assembles G1**, **Day 8 lights it + chooses**, **Day 9 pays
  off.**
- Note the clean split: **your tasks build the ship's way in; only the ship heals the
  friend.** The fish/comfort you provide keeps them going but never cures — which is the ache
  the whole design is built around.
